## 粒度指針

このテンプレートの各セクションで求められる粒度を以下に示す。

### 必須（省略不可）
- **作成/変更ファイル**: テーブル形式で全ファイル。箇条書き不可
- **テスト結果**: テストスイート名 + テスト数 + 全体リグレッション有無
- **テストケース一覧**: describe ブロック単位でケース名と期待値をテーブル化
- **予測との比較**: 3列テーブル（予測 / 実際 / 乖離）。箇条書き不可
- **意思決定ログ**: テーブル形式

### 条件付き必須
- **追加パッケージ**: パッケージ追加時のみ。なければセクション省略可
