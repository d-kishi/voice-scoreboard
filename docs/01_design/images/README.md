# 画面デザイン一覧

Figma Make で作成した画面デザインのスクリーンショット。
SDD 設計フェーズ（`spec-design`）での UI 参考資料として使用する。

## ファイル一覧

| ファイル | 画面 | 説明 |
|---------|------|------|
| `scoreboard-main.png` | メイン画面（IDLE） | ダーク背景、左右分割のスコア表示（大文字）、+1/-1ボタン、下部バーに音声入力/読み上げトグルとロールバック/リセットボタン |
| `scoreboard-listening.png` | 音声認識モード（LISTENING） | 画面中央にシアン発光リング付きマイクアイコン、「Ready」テキスト、カウントダウン表示。背景のスコアはディム表示 |
| `scoreboard-reset-confirm.png` | リセット確認ダイアログ | 背景ディム、中央にダイアログ（「スコアを 0-0 にリセットしますか？」）、キャンセル/リセット（赤）ボタン |
| `scoreboard-reset-on-gameset.png` | 試合終了画面 | ゴールドの「試合終了」バナー、リセットボタンのみ表示、+1/-1ボタンは非表示（操作無効） |
