# Voice Scoreboard

音声で操作できるスポーツスコアボードアプリ（Android）

## 特徴

- **音声でハンズフリー操作** — 「スコア」と呼びかけるだけで音声モードに切り替わり、「右」「左」で得点を加算
- **大きなスコア表示** — 横画面いっぱいに得点を表示。体育館の離れた場所からも確認可能
- **タッチ操作にも対応** — 音声が使えない環境でも +1 / -1 ボタンで操作可能
- **オフライン動作** — ネットワーク不要。体育館やグラウンドなど電波の弱い場所でも安定動作
- **自動ルール判定** — 25点マッチ・デュース（24-24以降は2点差決着）を自動判定し、試合終了時にホイッスル音で通知
- **画面常時点灯** — 試合中に画面が消えることがない

## 対応競技

| 競技 | ルール |
|------|--------|
| 6人制バレーボール | 25点マッチ・デュース対応 |

> 今後、9人制バレー・バスケ・卓球・バドミントン等への対応を予定しています。

## 音声コマンド

### 基本の流れ

```
「スコア」 → Ready音 → コマンド（右/左/ロールバック/リセット） → Roger → スコア読み上げ
```

1. **「スコア」** と発声してウェイクワードを起動
2. 「Ready」と音声応答し、マイクアイコン付きのリスニング画面が表示される（5秒間受付）
3. コマンドを発声すると「Roger」と応答し、操作を実行してスコアを読み上げる

### コマンド一覧

| コマンド | 動作 |
|---------|------|
| **右** | 画面右側のチームに1点加算 |
| **左** | 画面左側のチームに1点加算 |
| **ロールバック** | 直前の操作を取り消し |
| **リセット** | スコアを 0-0 に戻す（確認なしで即実行） |

> 5秒以内にコマンドが検出されない場合は、自動的に待機状態に戻ります。

## タッチ操作

| 操作 | 動作 |
|------|------|
| **+1 ボタン** | 該当チームに1点加算 |
| **-1 ボタン** | 該当チームから1点減算（誤操作修正用） |
| **ロールバック** | 直前の操作を取り消し |
| **リセット** | 確認ダイアログ後、スコアを 0-0 に戻す |

## 設定

画面下部のコントロールバーから切り替え可能です。

| 設定 | 説明 |
|------|------|
| **音声入力** | 音声認識の ON/OFF。OFF にするとウェイクワード検知を停止 |
| **読み上げ** | 音声読み上げの ON/OFF。OFF でも Ready 音とホイッスル音は再生される |

設定は自動保存され、次回起動時に復元されます。

## 推奨環境

| 項目 | 要件 |
|------|------|
| OS | Android 10 以上 |
| 端末 | マイク搭載の Android スマートフォン / タブレット |
| 画面の向き | 横画面（ランドスケープ）固定 |

> 音声認識はデバイスのオンデバイスモデルを使用します。初回利用時にオフライン音声データのダウンロードが必要な場合があります。

## インストール

### APK から直接インストール

2. Android 端末で APK を開いてインストール（「提供元不明のアプリ」の許可が必要）
